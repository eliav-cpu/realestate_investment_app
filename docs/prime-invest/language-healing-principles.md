# Prime Invest — Language Healing Principles

Updated: 2026-07-03

This document defines a permanent wording layer for Prime Invest and related EXEL / ATLAS work.

The goal is to convert weak, generic wording into richer, sharper, more human and more commercial Hebrew.

## Core idea

Do not write like a form.
Do not write like a generic lead page.
Do not write like AI.
Do not write like a legal disclaimer in the main headline.

Write like a strong Israeli operator who understands the client, the process, the people behind the product, and the next step.

## Main language pillars

1. Clear value
2. Proven people
3. Guided process
4. Direct access
5. Human Hebrew
6. Specific next step

## Prime core sentence

אנחנו לא שולחים אתכם לגאורגיה. אנחנו מביאים את גאורגיה אליכם — בעברית, בישראל, עם ליווי מלא מאפס עד 100.

## Preferred direction

Use language around:

- פגישת היכרות
- רכישה ישירה מהיזם
- יזמים עם קבלות בשטח
- ליווי ישראלי מלא
- פרויקטים שנבחרו למשקיע הישראלי
- דרך ברורה מאפס עד 100
- מי עומד מאחורי הפרויקט
- מה קורה אחרי החתימה
- לא מתחילים לבד מול מדינה זרה

## Avoid as main language

- בדיקת התאמה
- השאירו פרטים
- נציג יחזור אליכם
- תמונת עסקה
- לפתוח עסקה
- לראות מספרים
- מה הכסף פותח
- נפתח לכם תרחיש
- ניתן לבחון אפשרות
- פרויקט מדהים
- מיקום מעולה
- הזדמנות שלא תחזור
- שפה גנרית מדי
- שפה משפטית מדי

## Strong phrase pairs

לא קונים נכס בחו״ל בגלל תמונה יפה.
קונים אחרי שמבינים מי היזם, מה המיקום, איך עובד התהליך ומי מלווה אתכם.

היתרון הוא לא רק הנכס.
היתרון הוא הדרך שבה מגיעים אליו.

רכישה ישירה מהיזם לא אומרת לבד.
היא אומרת להגיע נכון, עם Prime לצדכם.

אם כבר רוכשים בגאורגיה — לא מתחילים לבד.
מתחילים עם מי שמכיר את היזמים, הפרויקטים והשטח.

## Operating rule

Every new customer-facing sentence should pass through this check:

1. Does it sound human?
2. Is it specific?
3. Is it commercial enough?
4. Does it build trust?
5. Does it avoid generic wording?
6. Does it make the next step clear?

If not, rewrite it.
